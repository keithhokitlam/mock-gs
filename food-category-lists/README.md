# Food Category Lists

Place your Excel documents here. They will be converted to JSON and displayed as searchable, sortable tables.

## Workflow

1. **Add your Excel file** (e.g. `Fruits 20260211.xlsx`) to this folder.
2. **Run** `npm run process-fruits` to convert it to `src/data/fruits.json`.
